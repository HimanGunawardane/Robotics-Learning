**Essential Commands**

-creating a package and workspace-

(create and enter a ROS workspace)
mkdir -p ~/<workspace name>/src
cd ~/<workspace name>


(create a new package)
ros2 pkg create --build-type ament_python --license Apache-2.0 <package_name>

(create a hello world! package)
ros2 pkg create --build-type ament_python --license Apache-2.0 --node-name my_node my_package



-building/compiling a package-


(builds all ROS2 packages)
colocn build

(build a specific package)
colcon build --packages-select <package name>



-sourcing and running a package-

(source package setup file(must be run in directory)ros2)
source install/local_setup.bash

(run a package)
ros2 run <package name> <node name>

(runs a ROS2 executable)
ros2 run <package_name> <executable_name>

(launch python scripts)
ros2 launch <nodename> <scripname.py>



-attributes of nodes and packages-

(used to set attributes of a node or package)
rqt

(shows graph of data flow bettween ROS2 nodes
ros2 run rqt_graph rqt_graph

(lists all nodes when ROS2 program is running)
ros2 node list

(get more info about a node)
ros2 node info <node_name>

(listen to the data on a specific topic)
ros2 topic echo <topic_name>

(list Ros paramiters)
ros2 param list

(get the value of a parramiter)
ros2 param get <node_name> <parameter_name>

(set parramiter)
ros2 param set <node_name> <parameter_name> <value>

(list all actions)
ros2 action list

(list action file path)
ros2 action list -t

(shows data types in interface actions)
ros2 interface show turtlesim/action/RotateAbsolute

(send action to the robot)
ros2 action send_goal <action_name> <action_type> <values>



**other commands**

(start ROS turlte sim)
ros2 run turtlesim turtlesim_node

(control turtle sim from a new terminal)
ros2 run turtlesim turtle_teleop_key

(rqt console. lists action completed)
ros2 run rqt_console rqt_console

(set minimum log message)
ros2 run turtlesim turtlesim_node --ros-args --log-level <log level>

(change which turtle robot is being controlled)
ros2 run turtlesim turtle_teleop_key --ros-args --remap turtle1/cmd_vel:=turtle2/cmd_vel

ros2 launch turtlesim multisim.launch.py




(record infromation and inputs from a topic)
ros2 bag record <topic_name>

(run recorded inputs from a bag file)
ros2 bag play subset

(record the ifnormation from a service)
ros2 bag record --service <service_names>

(record all the information from all services)
ros2 bag record --all-services
